# sql5 v1.13 版本說明（已完成）

## 版本資訊
- **版本**：1.13
- **日期**：2026-05-03
- **名稱**：DEFAULT VALUES 與 LIMIT 語法增強

## 新增功能

### 1. INSERT DEFAULT VALUES
- `INSERT INTO table DEFAULT VALUES` - 使用預設值插入

### 2. LIMIT 語法增強
- `LIMIT n, m` - 跳過前 n 列，回傳 m 列（MySQL 風格）
- 需支援 `LIMIT <offset>, <count>`

### 3. 欄位 DEFAULT 約束
- `CREATE TABLE t (id INTEGER DEFAULT 0)`
- `CREATE TABLE t (id TEXT DEFAULT 'unknown')`

### 4. 字串連接
- `||` 運算子（已在 v1.11 測試）
- `SELECT 'hello' || ' ' || 'world'`

## 使用方式

```sql
-- DEFAULT VALUES
CREATE TABLE t (id INTEGER DEFAULT 100, name TEXT DEFAULT 'unknown');
INSERT INTO t DEFAULT VALUES;
SELECT * FROM t;

-- LIMIT 語法
SELECT * FROM t LIMIT 5, 10;

-- DEFAULT 約束
CREATE TABLE users (id INTEGER PRIMARY KEY, name TEXT DEFAULT 'guest');
INSERT INTO users (id) VALUES (1);
```
